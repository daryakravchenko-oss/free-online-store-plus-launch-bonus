ФОНОВЫЕ КАРТИНКИ ДЛЯ ВТОРОГО ВАРИАНТА ХИРО (слайдер)
====================================================

Это отдельная папка для фона ВТОРОГО варианта хиро (секция id="hero-v2").
Фон первого варианта лежит в соседней папке img/hero/ — не путать.

Положи сюда картинки и назови их по порядку:

    1.webp
    2.webp
    3.webp
    ... (можно до 8)

(имя — 1, 2, 3 … ИЛИ hero-1, hero-2 …; расширение — любое из jpg / jpeg / webp / png)

Поведение точно как у первого варианта:
- 0 картинок  -> остаётся тёмный фон с золотым свечением;
- 1 картинка  -> статичный фон;
- 2 и больше  -> слайдшоу с плавным перекрёстным переходом (каждые 5 сек)
                 и лёгким "наездом" камеры (Ken Burns).

Рекомендации: широкие горизонтальные фото, минимум 1920×1080.
Этот файл можно не трогать, он держит папку в репозитории.
