ФОНОВЫЕ КАРТИНКИ ДЛЯ ХИРО (слайдер)
====================================

Положи сюда картинки для фона первого экрана и назови их по порядку:

    hero-1.jpg
    hero-2.jpg
    hero-3.jpg
    hero-4.jpg
    ... (можно до hero-8)

Форматы: jpg, jpeg, webp или png — слайдер подхватит любой из них автоматически
(имя — hero-1, hero-2 … ИЛИ просто 1, 2 …; расширение — любое из этих).

Поведение:
- 0 картинок  -> остаётся текущий тёмный фон с золотым свечением;
- 1 картинка  -> статичный фон;
- 2 и больше  -> слайдшоу с плавным перекрёстным переходом (каждые 5 сек) и
                 лёгким "наездом" камеры (эффект Ken Burns), как на
                 sellvia.com/order-processing-grant.

Рекомендации:
- широкие горизонтальные фото, минимум 1920×1080 (лучше 2400px по ширине);
- сюжет/лица держи ближе к верхней-центральной части кадра
  (фокус кадрирования — center 30%);
- поверх фото лежит тёмно-золотой оверлей для читаемости текста — это нормально.

Этот файл можно не трогать, он просто держит папку в репозитории.
